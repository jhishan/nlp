This is a node application

In this directory type in

'npm install'

This will install all the dependencies needed for this project

After that, you will need to get a token from a gmail account

You can do this by typing the following in the console

'node quickstart.js'

This will give you a link. Copy this link and enter it in your browser, here you will receive a token.
Paste this token back in the terminal and this will get you authenticated.

After that, type in

'node bin/www'

You are all set. Type in localhost:3000 in your browser and you can view your emails with sentiment values
